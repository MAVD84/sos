# SOS Pet App Blueprint

## Overview

SOS Pet is a Flutter application designed to help pet owners find their lost pets. Users can report a lost pet with details and a picture, and browse a list of lost pets reported by others.

## Style and Design

*   **Theming**: The app uses a Material 3 theme with a custom color scheme and typography from Google Fonts.
*   **Colors**:
    *   Primary: #F25C54 (Red)
    *   Secondary: #F2B90C (Yellow)
    *   Tertiary: #4C5958 (Gray)
*   **Fonts**:
    *   Display: Oswald
    *   Title: Roboto
    *   Body: Open Sans
*   **Layout**: The app uses a simple and intuitive layout with a list of lost pets on the home screen and a form to report a lost pet.

## Features

*   **Home Screen**: Displays a list of lost pets with their name and location.
*   **Report Pet Screen**: A form to report a lost pet with fields for pet name, description, and last seen location.
*   **Navigation**: The app uses `go_router` for navigation between screens.
